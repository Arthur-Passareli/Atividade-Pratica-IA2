# IA2 DevCard

O que é o projeto e descrição: DevCard é um projeto desenvolvido para a IA2 de Prática de Aplicações Móveis. A aplicação permite criar cartões personalizados de desenvolvedor de forma simples e prática, adicionando informações como nome, foto, tecnologias e redes sociais. O projeto possui uma tela inicial, uma página de preview para visualizar o cartão em tempo real, uma tela de cadastro para preenchimento das informações e uma página de sucesso confirmando a criação do DevCard.
